Detailed Description

The values stored in entities.

Each entity has a value for each property it is associated with. The value is of the type specified by the property and constrained by the property's settings.

◆ ikarus_value_is_equal()

IKA_API bool ikarus_value_is_equal ( IkarusEntityValue const *  left,
IkarusEntityValue const *  right 
)

Checks if two entity values are equal.

Two entity values are equal if they are of the same type and their value is considered equal. Note that floating point values can only be checked for approximate equality.

Parameters
leftThe left-hand entity value.
rightThe right-hand entity value.
Returns
True if the values are considered equal, false otherwise.
Remarks
Null values compare false to all other values. As do invalid values. Indeterminate values however, compare true to other indeterminate values of the same type.
